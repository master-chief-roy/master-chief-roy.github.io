# Portafolio Profesional – Ronny Fabricio Toctaquiza Piedra

Bienvenido a mi portafolio profesional. Aquí encontrarás una muestra de mis proyectos, experiencia, certificaciones y habilidades en ingeniería, software y redes.

## Características principales
- **Diseño moderno y responsive**: Adaptado a cualquier dispositivo.
- **Secciones**: Inicio, Sobre mí, Proyectos, Tecnologías, Certificaciones y Contacto.
- **Integración de proyectos reales**: Acceso a detalles, recursos y enlaces externos.
- **Animación tipo terminal**: Presentación interactiva en la sección de contacto.
- **Accesibilidad y SEO**: Imágenes con descripciones, meta etiquetas y navegación clara.

## Tecnologías utilizadas
- HTML5, CSS3 (con variables y media queries)
- JavaScript (SPA-like, manejo de datos dinámicos)
- Fuentes modernas y FontAwesome para iconos

## Cómo usar
1. Clona o descarga este repositorio.
2. Abre `index.html` en tu navegador favorito.
3. Navega por las secciones usando el panel lateral.

## Personalización
- Edita `data.json` para actualizar tu información personal, proyectos y redes sociales.
- Agrega tus imágenes en la carpeta `imagenes/` o `img/`.
- Modifica los estilos en `style.css` para adaptar la paleta de colores o el diseño.
