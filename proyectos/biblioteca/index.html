Sistema de Gestión Bibliotecaria

Para ejecutarla, abre una terminal en la carpeta del proyecto y ejecuta:

python main_biblioteca.py

Consulta el README.md para más detalles y ejemplos de uso.

$ python main_biblioteca.py
